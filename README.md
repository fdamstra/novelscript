# NovelScript

I was interested in showing that claude can create things that haven't existed before. This was my first idea: write a scripting langauge intended for use by AI.

The only files written by a human are this one and [prompt.txt], which is the initial prompt.

Additionally, select questions and answers were done in the conversations in [conversations/]. They are numerically ordered.
